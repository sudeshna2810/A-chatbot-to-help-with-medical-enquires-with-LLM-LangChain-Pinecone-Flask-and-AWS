# A-chatbot-to-help-with-medical-enquires-with-LLM-LangChain-Pinecone-Flask-and-AWS

# How to Star?
### Steps are as follows:

Clone the repository

```bash
git clonehttps://github.com/sudeshna2810/A-chatbot-to-help-with-medical-enquires-with-LLM-LangChain-Pinecone-Flask-and-AWS.git
```

### Step 1 -> After opening the Repository, create a conda environment

```bash
conda create -n healthbot python=3.10 -y
```

```bash
conda activate healthbot
```

### Step 2 -> Install the requirements
```bash
pip install -r requirements.txt
```
